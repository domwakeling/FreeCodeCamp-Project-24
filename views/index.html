Image Search Abstraction Layer

Start a search

To start a search, use the API https://doms-img.herokuapp.com/api/imagesearch/ followed by your search term — so for instance, if you want to search for images of skiing you should use

https://doms-img.herokuapp.com/api/imagesearch/skiing

You will receive a JSON array containing ten results.

To change the set of results that are returned, you may append ?offset= to the search API, together with the offset that you require. So if we wanted to see the next ten images of skiing, we would use

https://doms-img.herokuapp.com/api/imagesearch/skiing?offset=10

Search history

If, instead of specifying the search string, you instead send the blank API ...

https://doms-img.herokuapp.com/api/imagesearch/

... you will instead receive a JSON array containing details of the ten most recent searches that the service has provided.
